# Vue-mini-template

> 这是一个极简的 vue admin 管理后台。它只包含了 Element UI & vue-router，这些搭建后台必要的东西。
>
> 本项目基于 https://github.com/PanJiaChen/vue-admin-template/blob/master/README-zh.md， 但是在此之上删减了很多东西，因此会简单很多。

### 启动项目

```bash
npm i

npm run dev
```
